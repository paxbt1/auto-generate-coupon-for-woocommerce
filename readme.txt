= How it's Work? =
1. WooCommerce > Auto generate coupon for Woocommerce
2. Check "Activeate campaign"
3. Select "Discount type" : FIxed or Percent Discount
4. You can Choose "Individual use only"
5. Set "Discount amount"
6. You can choose "Allow free shipping"
7. Just Set "Coupon expiry date" as a simple String!
8. At the end fill "Message for New Coupon" text area with your Custom Message to present Coupon to user Notice that Your string should contain "%$",
It's replaced with coupon code.
9. Hit the "Save Changes".
